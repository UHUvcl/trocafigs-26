# Notas de aprendizado

Este projeto foi usado como prática de desenvolvimento web, produto digital e
publicação em produção.

## Aprendizados técnicos

- Estruturação de PWA sem framework.
- Uso de Service Worker para cache/offline.
- Organização de lógica em módulo testável (`core.mjs`).
- Manipulação de estado local com `localStorage`.
- Parser de texto flexível para entradas reais de usuário.
- Renderização dinâmica de listas e cards.
- Geração de mensagens prontas para WhatsApp.
- Deploy estático em Vercel e Netlify.
- Versionamento de assets para evitar cache antigo.
- Testes automatizados com Node.js.

## Aprendizados de produto

- Partir de um problema real observado.
- Validar fluxo para diferentes idades.
- Reduzir cadastro e fricção na primeira versão.
- Priorizar entrada rápida de dados.
- Separar recursos sociais para uma fase com mais segurança.

## Aprendizados de responsabilidade

- Evitar uso de marcas, escudos, fotos e materiais protegidos.
- Não coletar dados pessoais sem necessidade.
- Pensar em segurança antes de recursos envolvendo crianças e adolescentes.
- Deixar claro que o app é independente.
