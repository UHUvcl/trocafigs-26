# Roadmap

## Fase atual

PWA estático com:

- controle de figurinhas;
- faltantes;
- repetidas;
- listas para WhatsApp;
- comparação de trocas;
- backup/importação;
- funcionamento offline.

## Próximas melhorias de produto

- Melhorar refinamento visual com testes com usuários.
- Criar um modo pacote mais guiado.
- Melhorar entrada por voz com instruções claras.
- Criar importação de listas vindas de WhatsApp/planilha.
- Criar resumo de troca mais inteligente.
- Criar tela de progresso por seleção/confederação.

## Fase social segura

Antes de liberar recursos públicos, será necessário planejar:

- login;
- perfis;
- denúncia e bloqueio;
- moderação;
- regras para menores de idade;
- locais de troca seguros;
- consentimento dos responsáveis quando aplicável.

## Ideias futuras

- Grupos privados por código.
- Compartilhamento de álbum por link.
- Sincronização em nuvem.
- Leitura por câmera/OCR.
- Mapa de pontos de troca autorizados.
- Painel administrativo para eventos de troca.
